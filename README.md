## INSTALL

### Windows
```
Download VIPainter.(version).exe
Then run it to install

下载安装包 VIPainter.(版本号).exe 然后运行安装
```
### Linux

#### Build dependencies

* c++11
* qmake >= 5.2

#### Build
```
git clone https://github.com/VIPainter/VIPainter
cd VIPainter
mkdir build
cd build
qmake ..
make
```
